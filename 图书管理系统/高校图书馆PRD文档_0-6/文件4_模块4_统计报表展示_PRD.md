# 模块4：统计报表展示 PRD

## 4.1 需求概述

### 4.1.1 背景
数据驱动决策是现代图书馆管理的核心需求。通过排行榜和统计报表，可以洞察读者偏好、优化馆藏结构、评估服务效能。

### 4.1.2 目标
- 提供多维度的借阅数据分析
- 支持馆藏优化决策
- 生成标准化的月度/年度运营报表
- 实现数据可视化（CLI文本图表）

### 4.1.3 范围

| 包含 | 不包含 |
|------|--------|
| 借阅排行榜（TOP N） | 数据导出为Excel/PDF |
| 月度/年度统计报表 | 与BI系统对接 |
| 分类/院系/时间维度分析 | 预测性分析（AI推荐） |
| 文本图表展示 | 图形化界面图表 |

## 4.3 详细功能需求

### 4.3.1 借阅排行榜（FR-301 ~ FR-310）

#### FR-301 图书热度榜

| 排名维度 | 计算方式 | 更新频率 |
|---------|---------|---------|
| 总借阅次数 | 累计borrow_count | 实时 |
| 月度借阅 | 当月借阅次数 | 每月1日重置 |
| 季度借阅 | 当季借阅次数 | 每季度重置 |
| 年度借阅 | 当年借阅次数 | 每年重置 |

**展示字段**：排名、图书编号、书名、作者、分类、借阅次数、环比变化

#### FR-302 读者活跃榜

| 排名维度 | 计算方式 |
|---------|---------|
| 借阅总量 | 累计借阅本数 |
| 月度活跃 | 当月借阅本数 |
| 阅读广度 | 借阅不同分类数 |
| 准时归还率 | 按时归还次数/总归还次数 |

#### FR-303 分类畅销榜

```
按分类分组，计算每个分类的：
- 借阅总次数
- 借阅总人数
- 人均借阅量
- 图书流通率（借阅次数/总副本数）

排序：按借阅总次数降序
```

#### FR-304 新书推荐榜

- 出版年份≥当前年份-2
- 按借阅次数排序
- 排除已下架/损坏图书

### 4.3.2 月度统计报表（FR-311 ~ FR-320）

#### FR-311 报表结构

```
【XXXX年XX月 图书馆运营月报】
═══════════════════════════════════════

一、运营概况
├─ 新增读者: XX人
├─ 总借阅量: XX次
├─ 总归还量: XX次
├─ 当前借出: XX本
├─ 逾期率: XX.XX%
└─ 罚金收入: XX.XX元

二、分类借阅分析
[文本柱状图展示各分类借阅量]

三、读者行为分析
├─ 最活跃读者: XXX (XX本)
├─ 平均借阅周期: XX天
├─ 平均续借次数: X.X次
└─ 预约转化率: XX.XX%

四、财务统计
├─ 罚金收入: XX.XX元
├─ 赔偿收入: XX.XX元
└─ 合计: XX.XX元

五、热门图书TOP10
[排名列表]

六、运营建议
[基于数据的自动建议文本]
```

#### FR-312 数据计算逻辑

| 指标 | 计算公式 |
|------|---------|
| 新增读者 | COUNT(register_time in 本月) |
| 总借阅量 | COUNT(borrow_time in 本月) |
| 总归还量 | COUNT(return_time in 本月 AND status=RETURNED) |
| 逾期率 | COUNT(逾期记录) / COUNT(本月应还记录) × 100% |
| 图书流通率 | 总借阅次数 / 总馆藏册数 × 100% |
| 人均借阅 | 总借阅量 / 活跃读者数 |

### 4.3.3 年度统计报表（FR-321 ~ FR-330）

#### FR-321 年度对比分析

| 对比维度 | 说明 |
|---------|------|
| 同比 | 今年 vs 去年同月 |
| 环比 | 本月 vs 上月 |
| 均值 | 今年月均 vs 去年月均 |

#### FR-322 趋势预测（简化）

基于近12个月数据，使用简单移动平均：
```
下月预测 = (近3月均值 × 0.5) + (近6月均值 × 0.3) + (近12月均值 × 0.2)
```

---

### 4.3.4 专项分析（FR-331 ~ FR-340）

#### FR-331 院系借阅分析

| 分析项 | 说明 |
|-------|------|
| 院系借阅排名 | 按总借阅量排序 |
| 人均借阅 | 该院系总借阅 / 该院系读者数 |
| 偏好分类 | 该院系借阅最多的分类 |
| 活跃时段 | 该院系读者借阅高峰时段 |

#### FR-332 时段借阅分析

```
按小时段统计借阅量：
- 上午 (8:00-12:00)
- 下午 (12:00-18:00)
- 晚上 (18:00-22:00)

识别高峰时段，优化开馆时间和人员配置
```

## 4.4 非功能需求

### 4.4.1 性能需求

| 指标 | 目标值 |
|------|--------|
| 月度报表生成 | ≤3秒（10000条记录） |
| 排行榜查询 | ≤1秒 |
| 年度报表生成 | ≤5秒 |

### 4.4.2 数据准确性

- 统计口径一致：所有报表使用同一数据源
- 时间边界清晰：月度统计按自然月，年度按自然年
- 去重处理：同一读者同一本书多次借阅分别计数
