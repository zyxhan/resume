# 核心模块 · 画面叙事写作法

> 这不是"一种模板"。这是这个 Skill 的写作底层逻辑。所有文章类型——随笔、日记、读书笔记、抖音文案——都默认用这个方式写。

## 核心理念

**不要说"我很孤独"。写地铁玻璃窗上划过的雨水。**

情绪不直接说。情绪藏在画面里、对话里、具体的物件里。读者自己会感受到——那比你直接告诉他要强烈一百倍。

## 十条法则

### 1. 开头即画面

不要从"最近我想了很多关于XX的事"开始。从一个具体的时刻开始——

> ❌ "大学让我变了很多。"
>
> ✅ "兰州的天空阴沉了一整天，可还是燥热难捱。我躲在窗台，微弱的凉风从窗缝里溜进来。"

> ❌ "时间过得真快，一转眼大一就结束了。"
>
> ✅ "去年六月，高考结束那晚，我和同学在网吧通宵。K 买了一排红牛，说今晚谁都别想睡。"

**第一句话就应该让读者看见一个场景。**

### 2. 具体 > 抽象

永远用具体的东西替代抽象的概念。

| 不要说 | 写成 |
|--------|------|
| 我们关系很好 | 每晚放学我两都一起骑车回家 |
| 我很焦虑 | 凌晨三点躺在床上，盯着天花板上那盏没关的灯 |
| 青春回不去了 | 我们开上了更好的车，却开始怀念那辆四十多码的电驴 |
| 我很孤独 | 我在联系人列表里点开一个又一个名字，最后还是划掉了 |
| 她很敏感 | 江枂突然变得紧张，不安地讲了好多好多话 |

### 3. 用真名实姓

一点点奶茶、十六号线、鲁迅公园、兰山、《龙族》、汪苏泷——真实的地名、品牌、书名、歌名自带质感。不要说"一家奶茶店"，说"一点点"；不要说"一首歌"，说《那个男孩》。

### 4. 对话是活的

对话推动叙事，比平铺直叙有力得多。写对话时——

- 让它像真的在说话，不是"文学腔"
- "卧槽"、"狗儿子才会想你"、"你小子被打脸了吧"——保留口语
- 对话里藏着情绪比叙述里多

> "干啥呢？"
>
> "忙着呢，我这才工作几天就想我了？"

这两句对话透露出的东西，比"我们是很好的朋友"多一百倍。

### 5. 短段呼吸

一个句子就是一段。不用填满。留白让读者自己感受。

> 我哂然一笑，如山岳般压在心头的俗世纷扰在这一刻很轻易地被我抛之脑后。
>
> 一切早已不重要了。
>
> 现在我只想安然孤坐在此，一人赏尽属于我的这一整个秋。

### 6. 锚点

每篇文章有一个"锚"——一个具体的人（K、江枂）、一件物（电驴、一点点奶茶）、一个地点（兰山顶、鲁迅公园长椅）。文章绕着这个锚转，不散。

你写时间的随笔——锚点可以是"网吧通宵那晚"、"高三教室那个座位"、"大一宿舍第一晚的天花板"。选一个，所有的情绪都挂在这一个东西上。

### 7. 时间跳跃

在"那时"和"现在"之间来回切换，制造层次感——

> "现在回想起来…"
> "当时只道是寻常呀。"
> "许多年后…可每次谈及…"

不按时间顺序写。从"现在"看"过去"，或者从"过去"滑到"现在"。

### 8. 引用是情感的快捷键

《龙族》里路明非的犹豫、汪苏泷的那句歌词——引用不是在掉书袋，是在说"你知道那个感觉吗？就是那个"。引用是和读者之间的暗号。

### 9. 结尾不总结

不要说"所以我们应该珍惜时间"。不总结，不说教。可以——

- 留一个画面： *"秋风轻拂，万千落枫在我眼前纷飞，像一场金红色的雨。"*
- 留一句话： *"我不想释怀。"*
- 留一个动作： *"我给 K 发去消息。'干啥呢？'"*

戛然而止比画蛇添足好一万倍。

### 10. 写的时候像在跟朋友说话

折刀文章最大的特点是——你读起来像是在听一个人跟你聊天。不是"作家在写作"，是"朋友在讲故事"。语气自然、会自嘲、会说脏话、会突然认真又突然开玩笑。这就是最好的声音。

## 写作时的工作流

当用户说"我想写XX"时，按以下步骤引导：

**Step 1 — 找锚点**
"这件事里，最让你印象深刻的画面是什么？一个具体的东西——一个人、一个地方、一个物品。"

**Step 2 — 从画面开头**
"就从那个画面开始写。不用铺垫，不用介绍背景。直接写那个画面。"

**Step 3 — 跟着感觉走**
用户写的过程中，不干预结构。只在他卡住时问：*"然后呢？你接下来看到了什么？听到了什么？"*

**Step 4 — 写完后再看**
用户写完初稿后，对照"十条法则" + "五件武器"，逐条指出哪里做得好、哪里可以加强。不要改他的句子，只标注位置和方向。优先参照"你的武器"——因为那是他自己已经验证过的技巧。

**Step 5 — 用户自己改**
给建议，不给替代方案。让他自己改。

## 初稿 vs 润色示例

以下示例展示"画面叙事法则"如何应用——

**初稿（说情绪）：**
> 大一这一年我感觉自己变化很大。高中的时候什么都不用想，现在要面对很多新的事情，有时候觉得很迷茫。时间真的过得很快，一转眼就过去了。

**润色（画面叙事）：**
> 前两天收拾行李准备回家，翻到了高考准考证。照片上的我穿着校服，头发剪得很短，眼神里有一种现在不会有的东西。
>
> 我把准考证夹回书里，没扔。

# 你的武器 · 来自你自己的文章

> 以下技巧不是从别人那里学的——是从你自己那篇《小安》里提炼出来的。你已经用过它们，而且用得很好（6000 播放、336 赞证明了这一点）。以后写作时，有意识地调用这些武器。

## 武器一 · 第三人称写自己

**你不是"我"。你是"小安"。**

这篇最厉害的叙事选择——你不以受害者身份倾诉，你拉开距离，让自己被注视。

| 第一人称（倾诉） | 第三人称（注视） |
|------------------|------------------|
| "我很难过" | "她浑身湿透，眼底藏着说不清的落寞" |
| 读者感觉你在诉苦 | 读者感觉自己在目睹 |
| 情绪是你说出来的 | 情绪是读者自己感受到的 |

**以后怎么写**：当你想写自己的经历时，问问自己——能不能把"我"换成"她/他"？能不能让自己被另一个人注视？距离产生力量。倾诉是说服别人，注视是让别人自己走进去。

## 武器二 · 童年锚点画面

**姥爷踹翻大理石餐桌。**

全文最重的一句话。它之后不需要解释——不需要说"所以她有童年阴影"、"所以家庭关系破裂"。读者看到这个画面，什么都懂了。

**以后怎么写**：写一个人的创伤或成长时，不要写"历史"——找一个**具体的画面**。一个就够了。那个画面要满足三个条件：
1. 真实发生过
2. 你闭上眼睛就能看见
3. 不需要上下文也能让读者感到冲击

找到了，就放在那里。不用解释。

## 武器三 · 双线叙述

**一条"现在"的线 + 一条"过去"的线，交替推进。**

| 现在的线 | 过去的线 |
|---------|---------|
| 晚上八点，电话响了 | 育红班开始就是留守儿童 |
| 她说"我不行了" | 父母离婚、姥爷踹翻餐桌 |
| 她在国道上走 | 父亲只会转生活费 |
| 凌晨一点去接她 | |
| 她住在我家 | |
| 凌晨她给奶奶打电话 | |
| 她冲进雨里 | |
| 她回家了 | 如今已经上大学了 |

两条线交替——现在线往前走一步，就切一块"为什么她会这样"的童年拼图。你不是一口气倒完所有的背景，而是**像剥洋葱一样，一层现在、一层过去**。

**以后怎么写**：写一个"正在发生的事"时，每推进一两个场景，就穿插一小段"为什么会这样"的过去。现在的画面和过去的画面像绳子一样拧在一起，读者的情绪会跟着越拧越紧。

## 武器四 · 最重的情緒用最轻的笔

**"最后只泡了一桶泡面，安静地吃着。"**

这句话里的克制，比任何"她太惨了"都有力量。

她离家出走、在雨夜里走了三个小时、被狗追赶、差点没大学上——然后你写的是"泡了一桶泡面"。

**以后怎么写**：检查你的文章，找到情绪最重的段落。那一段里，有没有"我好难受"、"我觉得很委屈"之类直接说情绪的词？删掉。换成**一个安静的动作**——吃泡面、看窗外、叠衣服、发呆。越是重的情緒，越不要说出来，压在一个平凡的画面里。读者自己会感受到重量。

## 武器五 · 结尾不安慰

**"我也衷心希望如此。"**

不说"一切都会好起来的"。不说"她终于获得了幸福"。就说"我希望"。像一个朋友在远处看了一眼，轻轻说了句祝福，然后不打扰。

这个结尾之所以好，是因为它**不替你解决问题**。生活的残酷在那里，善意也在那里——不需要强行升华。

**以后怎么写**：结尾检查——你是不是在"总结"、"升华"、"安慰"？是的话，删掉重写。结尾可以：
- **留一个画面**（"她冲进大雨里"）
- **留一句希望**（"我也衷心希望如此"）
- **留一个不确定**（"日子应该会好起来吧"）

但不要替你的人物（替你自己）解决问题。生活没解决的问题，文章也不应该替它解决。

## 武器六 · 一句话定调

**"我的内心是一座回音壁。" / "我不是在躲着活，我是在看着活。" / "我把这称之为天赋。"**

你有一种天赋——开头就能用一个精准的比喻或判断句，把整篇文章的魂钉住。读者看到这句话，就不想走了。

这不是"标题党"技巧。这是你对自己内心的准确命名。当你找到那个词、那句话，文章的调子就定了——后面写什么，读者都愿意跟着。

**以后怎么写**：动笔之前先问自己——"这件事如果只用一个比喻形容，是什么？"找到了，就把它放在开头。那句话一出来，后面的东西自己就会流。

## 武器七 · 真诚到没有包装

**"在这里先和你们说对不起，也希望你们可以体谅我一下，谢谢。"**

这是你那篇失眠的结尾。没有任何文学技巧——就是直接对着读者说话。但正因为没有任何包装，它是整篇最有力量的一句。

你写"真诚打动读者"——这句话本身就是最好的示范。有时候句子越简单、越直接、越不加修饰，越能打进人心里。不必每篇都用画面藏情绪。有些话，直接说。

**以后怎么写**：文章快结尾时，问问自己——"如果现在对面坐着一个朋友，我会怎么跟他说？"把最想说的那句话直接写出来，不加修饰，不加美化。就是那句。

## 你的核心信念

> "我希望用我的真诚打动读者。"

这是你告诉我的。记住它。画面叙事、五件武器、十条法则——这些都只是帮你把真诚**传递出去**的工具。真诚本身才是力量。工具可以学，真诚是你自己的东西，谁也不缺但谁也偷不走。

# 写作类型

## 随笔 · 核心类型

默认使用画面叙事法。开头就是一个具体的时刻、场景或对话。标题不用太正式，"时间是个很奇怪的东西"比"关于时间的思考"好一百倍。

## 日记 · 与自己对话

```
# [日期] · [一个词/一句话]

[今天一个具体的画面或瞬间]

[由这个画面想到的——不要求逻辑，自由流]
```

日记也不是流水账。挑"今天让我心里动了一下的那件事"就够。

## 读书笔记 · 与书的对话

```
# 📖 [书名] · [作者]

[这本书让我想起的一个画面/场景]

## 这本书在讲什么
（用你自己的话，不是目录）

## 三个击中我的点
1. **点1** — 哪个场景/情境触动了你？
2. **点2**
3. **点3**

## 我不认同的 / 没懂的地方

## 看完后我变了什么
[具体的，不是"格局变大了"——比如"开始注意到______"]
```

## 复盘反思

```
# 🔄 复盘：[事件]

## 预期 vs 现实
| 我以为 | 实际发生 |
|--------|---------|
| | |

## 关键转折点
哪个时刻/哪个决定改变了走向？

## 如果重来
- 保持______
- 改变______

## 一句话带走
> 一个具体的原则
```

## 书信

不用模板。像跟那个人面对面说话一样写。

## 诗歌/短句

- 分行就是节奏。短行快，长行慢。
- 意象 > 形容词。"风把路灯吹得发抖" > "晚风很凉"。
- 写完读一遍——不顺嘴的地方一定有问题。

## 短评

```
# [事件/现象]

## 第一反应
（不加修饰的真实想法）

## 换个角度
站在对立面看

## 一句话态度
```

# 抖音发布适配 🎬

## 文案结构

**画面型**（从随笔里截取最打动人的一个画面）：
```
[一个具体的画面/瞬间，2-3 句]

[为什么这个画面让我停下来了——1 句话]

[你觉得呢？/你也有过这种感觉吗？]
```

**故事型**：
```
[悬念开头：一个反常的/具体的细节]
[转折：但是/直到/后来我才发现——]
[感悟：就一句话]
```

## 标题法则

| 法则 | 例子 |
|------|------|
| 制造缺口 | "大多数人搞错了 XX 的真正意思" |
| 反差前置 | "我学了三年编程，最受用的是一句废话" |
| 情绪召唤 | "内向的人，看完这条你会释怀" |
| 具体数字 | "写了 47 篇随笔后发现的 3 件事" |

## 改写指令

- **"压缩成抖音文案"** → 保留核心画面/情绪，精简到 100-200 字
- **"拆成系列"** → 一篇长文拆成 3-5 条短内容
- **"提炼金句"** → 从文章里抽 3-5 句适合独立传播的句子

## 抖音排版规则

一句一行。大量空行。不堆在一起。加粗关键句。

```
**标题就是封面**

正文一句一行。
手机屏幕上看，
堆在一起就是一坨。

---
> 金句截图发抖音特别好看。
```

# INFP 写作心法 🌿

## 你的天然优势

| 擅长 | 写法 |
|------|------|
| 细腻感受力 | 写具体的瞬间，不写抽象道理 |
| 对"意义"的在乎 | 不回避追问，这正是你的深度 |
| 内心独白 | 把自我怀疑写出声音——最共鸣的东西 |
| 理想主义 | 别怕"矫情"，真诚的理想主义极稀缺 |

## 常见坑

| 坑 | 表现 | 解法 |
|----|------|------|
| 过度内省 | 写完不敢发 | 写完 10 分钟内必须发 |
| 完美拖延 | 开头改五遍 | 先发再改 |
| 情绪过载 | 没呼吸点 | 一两句轻的穿插在重的之间 |
| 逃避结论 | "也许吧""谁知道呢" | 可以有犹豫，但不能没态度 |

## 没状态时

1. **先听一首歌** — 用音乐把自己调到"有话说"
2. **从"今天我注意到"开始** — 不预设主题
3. **写给一个具体的人** — 真实写作都有倾诉对象
4. **允许自己写得烂** — 第一稿的任务只是存在

# 灵感激发系统 💡

| 你说 | 我给你 |
|------|--------|
| "没灵感" | 基于你的兴趣随机抛选题 |
| "给我一个开头" | 一个可以直接续写的句子 |
| "今天有什么可写的" | 从日期/天气/校园日常挖掘 |
| 发图片/描述场景 | 联想写作：画面里有什么 → 这让你想起什么 |

## 选题库

**技术 × 人文**
- "代码教会我的不是编程，是______"
- "在 C 语言里发现的人生隐喻"

# 写作原则

1. **画面先于情绪** — 写画面，不写形容词
2. **先完成再完美** — 第一稿只负责存在
3. **发了比写得好重要** — 完美是 INFP 最大的敌人
4. **你的弱点是你的风格** — 笨拙、犹豫、自我怀疑——写出来反而动人
5. **像跟朋友聊天一样写** — 最好的语气是你自己的语气
6. **结尾不总结** — 戛然而止比画蛇添足好
