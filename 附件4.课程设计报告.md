# 1 设计内容及要求

## 1.1 设计内容

随着高校图书馆馆藏规模的不断扩大，传统的人工借阅管理方式效率低下、容易出错，难以满足师生日益增长的借阅需求。读者借阅权限差异化配置、逾期罚金自动计算、图书快速检索等功能亟待信息化改造。在此背景下，开发一套图书馆智能借阅管理系统，实现图书借阅全流程的自动化管理，具有重要的现实意义与应用价值。

本课程设计采用 C 语言（C99 标准）开发了一套运行于 Windows CLI 命令行终端的图书馆智能借阅管理系统，可对读者信息、图书信息、借阅记录进行高效管理。系统基于四层架构（表现层、业务层、数据层、持久化层），划分为读者分类管理、图书借还管理、逾期罚金管理、统计报表展示、智能检索系统和数据持久化六大功能模块，支持图书 ≤ 20,000 种、读者 ≤ 10,000 人、借阅记录 ≤ 50,000 条的数据规模。本人主要负责模块5（智能检索系统）的设计与实现。

# 2 概要设计

## 2.1 需求分析

### 2.1.1 用户界面需求

图书馆智能借阅管理系统应提供结构清晰、操作简便的 CLI 命令行菜单界面，使用户一目了然。界面应采用 Unicode 双线框字符绘制边框，以 ANSI 24bit 真彩色区分功能区域和状态信息。系统应在操作前和操作后显示明确的提示信息，方便用户确认操作结果。

### 2.1.2 系统功能需求

图书馆智能借阅管理系统的目标是实现图书借阅全流程的信息化管理，系统共划分六大功能模块。各模块通过全局状态单例 `SystemState` 共享图书数组、读者数组、借阅记录链表和索引结构。系统功能应具备以下六点：
（1）读者分类管理功能：支持本科生、研究生、教师、职工四种身份的注册与权限配置；
（2）图书借还管理功能：完整借阅/归还/续借流程与多副本库存管理；
（3）逾期罚金管理功能：自动逾期检测、罚金计算、催还与缴纳；
（4）统计报表展示功能：多维度排行榜与文本可视化；
（5）智能检索系统功能：模糊检索、组合检索、分类浏览、排序与推荐（本人负责）；
（6）数据持久化功能：二进制文件序列化、自动备份与恢复。

### 2.1.3 系统性能需求

（1）为保证系统数据安全，所有数据在退出时自动持久化保存，支持定时自动备份；
（2）系统应支持图书 ≤ 20,000 种、读者 ≤ 10,000 人、借阅记录 ≤ 50,000 条的数据规模；
（3）单次借阅操作响应时间 ≤ 1.5 秒，检索操作响应 ≤ 1 秒（万条数据），报表生成 ≤ 3 秒；
（4）检索召回率 ≥ 90%，精确检索准确率 100%，权限验证准确率 100%；
（5）数据持久化成功率 ≥ 99.99%。

## 2.2 功能模块

根据需求分析中的六个主要功能，分别规划了以下六个子模块：

（1）模块1 —— 读者分类管理模块。此模块负责读者注册、按读者 ID 或姓名查询、读者状态管理（正常/挂失/注销）、读者类型变更。四种身份对应差异化权限：本科生 5本/30天/1次续借，研究生 10本/60天/2次续借，教师 15本/90天/3次续借，职工 5本/30天/1次续借。数据存储在 `kvec` 动态数组中，以 `khash` 哈希表建立读者 ID 快速索引。

（2）模块2 —— 图书借还管理模块。此模块是系统核心业务流程，负责图书增删改查、多副本库存管理。借阅时自动校验四项资格（读者挂失/注销、借满、逾期未还、欠费），归还时自动触发逾期检测并生成罚金记录。数据以 `kvec` 动态数组存储图书，ISBN 通过 `khash` 哈希表快速定位。

（3）模块3 —— 逾期罚金管理模块。此模块遍历所有借阅记录检测逾期，按 0.1 元/本/天的标准计算罚金（精确到分），支持催还通知、罚金缴纳和特殊情况罚金豁免。系统配置中提供宽限期设置（默认 3 天）。

（4）模块4 —— 统计报表展示模块。此模块提供借阅热度排行榜（日/周/月/年 TOP 20）、分类图书统计、读者活跃度排名、逾期率分析，统计结果以文本字符柱状图可视化呈现。

（5）模块5 —— 智能检索系统模块。此模块提供基础模糊检索（书名/作者关键字）、高级组合检索（书名+作者+分类+出版社 AND 逻辑）、分类浏览（分页展示）、多维度排序（书名/作者/热度/出版日期，升降序可选）和热门图书 Top-N 推荐（基于累计借阅次数降序）。本人负责此模块的全部代码编写。

（6）模块6 —— 数据持久化模块。此模块负责全量数据保存/加载（5 个二进制 .dat 文件）、退出自动保存、定时自动备份（可配置间隔）、手动备份/恢复至指定目录、操作日志记录与查询、系统配置管理。

## 3.1 系统主程序的实现

（1）设计过程及步骤：

系统主程序 `main.c` 是程序入口，需完成控制台编码设置、系统初始化、主菜单循环和退出清理四个阶段的串联。首先调用 Windows API 将控制台代码页设为 UTF-8 以避免中文乱码，然后依次初始化全局状态单例、加载持久化数据，进入主菜单无限循环，程序退出时自动保存数据并清理资源。

（2）程序代码及注释：

```c
/**
 * @file main.c
 * @brief 高校图书馆智能借阅管理系统 —— 程序入口
 */
#include <stdio.h>
#include <stdlib.h>
#ifdef _WIN32
#include <windows.h>
#endif
#include "common.h"
#include "data_types.h"
#include "menu_cli.h"
#include "module6_persist.h"
#include "cli_theme.h"

/* 设置控制台为 UTF-8 编码 */
static void setup_console_utf8(void) {
#ifdef _WIN32
    SetConsoleOutputCP(65001);   /* CP_UTF8 */
    SetConsoleCP(65001);
    setvbuf(stdout, NULL, _IONBF, 0);  /* 禁用行缓冲 */
#endif
}

/* 系统初始化 */
static int system_init(void) {
    ErrorCode err;
    err = system_state_init();     /* 初始化全局状态 */
    if (err != ERR_OK) return -1;
    err = persist_load_all();      /* 加载持久化数据 */
    if (err != ERR_OK)
        cli_info("未找到已有数据文件，将以空数据启动。");
    return 0;
}

/* 系统退出清理 */
static void system_cleanup(void) {
    SystemState *state = get_system_state();
    if (state->config.backup_on_exit)
        persist_backup(NULL);      /* 退出前自动备份 */
    persist_save_all();
    system_state_destroy();
}

int main(void) {
    setup_console_utf8();          /* 步骤1: 编码设置 */
    cli_welcome();                 /* 步骤2: 显示欢迎界面 */
    if (system_init() != 0)        /* 步骤3: 系统初始化 */
        return EXIT_FAILURE;
    menu_main();                   /* 步骤4: 进入主菜单循环 */
    system_cleanup();              /* 步骤5: 退出清理 */
    cli_goodbye();
    return EXIT_SUCCESS;
}
```

## 3.4 逾期罚金管理模块的实现

（1）设计过程及步骤：

通过对模块3的功能要求分析，该部分需遍历借阅记录链表，对比应还日期与当前系统日期，找出所有逾期且未归还的记录。罚金按 0.1 元/本/天计算，精确到分。宽限期（默认 3 天）内不计罚金。催还通知功能列出所有逾期记录及其关联读者信息，罚金缴纳功能支持全额和部分缴纳。

（2）程序代码及注释：

```c
/* 逾期检测核心逻辑 */
void detect_overdue(SystemState *state) {
    time_t now = time(NULL);
    for (size_t i = 0; i < kv_size(state->records); i++) {
        BorrowRecord *r = &kv_A(state->records, i);
        if (r->status == 0) {  /* 未归还 */
            struct tm due_tm = {0};
            sscanf(r->due_date, "%d-%d-%d",
                   &due_tm.tm_year, &due_tm.tm_mon, &due_tm.tm_mday);
            due_tm.tm_year -= 1900;
            due_tm.tm_mon  -= 1;
            time_t due_time = mktime(&due_tm);
            int days_overdue = (int)(difftime(now, due_time) / 86400);
            if (days_overdue > state->config.grace_period) {  /* 超出宽限期 */
                r->status = 2;  /* 标记逾期 */
                r->fine_amount = (days_overdue - state->config.grace_period)
                                 * state->config.fine_per_day;
            }
        }
    }
}
```

## 3.6 智能检索系统模块的实现

（1）设计过程及步骤：

通过对模块5的功能要求分析（本模块由本人 —— 朱轶涵负责设计与实现），程序需要实现以下功能：第一，大小写不敏感的模糊子串匹配算法，对书名、作者、出版社、分类等文本字段逐字符比对；第二，高级组合检索时对四个条件字段做 AND 逻辑组合，NULL 条件视为"不限"；第三，每次检索结果缓存到静态数组 `last_results[]`，后续排序操作直接对缓存调用标准库 `qsort`；第四，格式化表格输出，使用 ANSI 彩色控制序列实现隔行变色和状态色彩标识。

源文件位于 `src/module5_search/module5_search.c`（约 506 行），头文件位于 `include/module5_search.h`。代码文件头部标注 "作者: 朱轶涵"。

（2）程序代码及注释：

```c
/**
 * @file module5_search.c
 * @brief 模块5 —— 智能检索系统 (完整实现)
 * 作者: 朱轶涵
 */
#include "module5_search.h"
#include "cli_theme.h"
#include <ctype.h>

/* ---- 内部常量 ---- */
#define MAX_RESULTS  500        /* 最大缓存结果数 */

/* ---- 内部状态: 缓存最近一次检索结果 ---- */
static Book last_results[MAX_RESULTS];
static int  last_count = 0;
static int  demo_loaded = 0;

/* ---- 前向声明 ---- */
static void init_demo_books(void);
static int  fuzzy_contains(const char *text, const char *keyword);
static void print_result_table(const Book *books, int count,
                                int page, int per_page);

/* ============================================================
 *  辅助: 大小写不敏感子串匹配
 * ============================================================ */
static int fuzzy_contains(const char *text, const char *keyword) {
    if (!text || !keyword) return 0;
    if (!*keyword) return 1;
    int tlen = (int)strlen(text);
    int klen = (int)strlen(keyword);
    if (klen > tlen) return 0;
    for (int i = 0; i <= tlen - klen; i++) {
        int match = 1;
        for (int j = 0; j < klen; j++) {
            char tc = (char)tolower((unsigned char)text[i + j]);
            char kc = (char)tolower((unsigned char)keyword[j]);
            if (tc != kc) { match = 0; break; }
        }
        if (match) return 1;
    }
    return 0;
}

/* ============================================================
 *  search_basic() —— 基础检索
 *  按书名/作者关键字模糊匹配
 * ============================================================ */
ErrorCode search_basic(const char *keyword, int *out_count) {
    if (out_count) *out_count = 0;
    if (!keyword || !*keyword) return ERR_INVALID_PARAM;
    init_demo_books();
    SystemState *state = get_system_state();
    last_count = 0;
    for (size_t i = 0; i < kv_size(state->books) && last_count < MAX_RESULTS; i++) {
        Book *b = &kv_A(state->books, i);
        if (fuzzy_contains(b->title, keyword) ||
            fuzzy_contains(b->author, keyword))
            last_results[last_count++] = *b;
    }
    if (out_count) *out_count = last_count;
    return last_count > 0 ? ERR_OK : ERR_NOT_FOUND;
}

/* ============================================================
 *  search_advanced() —— 高级组合检索
 *  多字段 AND 逻辑: 书名 + 作者 + 分类 + 出版社
 * ============================================================ */
ErrorCode search_advanced(const char *title_keyword,
                          const char *author_keyword,
                          const char *category,
                          const char *publisher,
                          int *out_count) {
    if (out_count) *out_count = 0;
    init_demo_books();
    SystemState *state = get_system_state();
    last_count = 0;
    for (size_t i = 0; i < kv_size(state->books) && last_count < MAX_RESULTS; i++) {
        Book *b = &kv_A(state->books, i);
        /* NULL 条件视为"不限制"，跳过该字段匹配 */
        int ok_title     = (!title_keyword  || fuzzy_contains(b->title,     title_keyword));
        int ok_author    = (!author_keyword || fuzzy_contains(b->author,    author_keyword));
        int ok_category  = (!category       || fuzzy_contains(b->category,  category));
        int ok_publisher = (!publisher      || fuzzy_contains(b->publisher, publisher));
        if (ok_title && ok_author && ok_category && ok_publisher)
            last_results[last_count++] = *b;
    }
    if (out_count) *out_count = last_count;
    return last_count > 0 ? ERR_OK : ERR_NOT_FOUND;
}

/* ============================================================
 *  search_sort_results() —— 结果排序
 *  sort_field: 1=书名 2=作者 3=热度 4=出版日期
 * ============================================================ */
ErrorCode search_sort_results(int sort_field, int ascending) {
    if (last_count <= 1) return ERR_OK;
    int (*cmp)(const void*, const void*) = NULL;
    switch (sort_field) {
    case 1: cmp = ascending ? cmp_by_title_asc  : cmp_by_title_desc;  break;
    case 2: cmp = ascending ? cmp_by_author_asc : cmp_by_author_desc; break;
    case 3: cmp = ascending ? cmp_by_hotness_asc : cmp_by_hotness_desc; break;
    case 4: cmp = ascending ? cmp_by_date_asc   : cmp_by_date_desc;   break;
    default: return ERR_INVALID_PARAM;
    }
    qsort(last_results, last_count, sizeof(Book), cmp);
    print_result_table(last_results, last_count, 1, last_count);
    return ERR_OK;
}

/* ============================================================
 *  search_hot_recommend() —— 热门图书推荐 (基于借阅热度)
 * ============================================================ */
ErrorCode search_hot_recommend(int top_n, int *out_count) {
    if (out_count) *out_count = 0;
    if (top_n < 1) top_n = 10;
    init_demo_books();
    SystemState *state = get_system_state();
    int total = (int)kv_size(state->books);
    if (total > MAX_RESULTS) total = MAX_RESULTS;
    for (int i = 0; i < total; i++)
        last_results[i] = kv_A(state->books, i);
    qsort(last_results, total, sizeof(Book), cmp_by_hotness_desc);
    last_count = (total < top_n) ? total : top_n;
    if (out_count) *out_count = last_count;
    return ERR_OK;
}
```
