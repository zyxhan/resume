## Task 1：网络编程（2月-5月）【必做】

### 核心知识点

**网络基础**
- TCP/IP四层模型
- TCP三次握手/四次挥手（面试必问）
- TCP vs UDP区别（面试必问）
- IP地址、端口、字节序（大端/小端）

**Socket编程**
- TCP Socket：`socket` → `bind` → `listen` → `accept` → `recv`/`send`
- UDP Socket：`socket` → `bind` → `recvfrom`/`sendto`
- 阻塞 vs 非阻塞IO

**IO多路复用（重点）**
- `select`：基本原理，fd_set操作
- `poll`：和select的区别
- `epoll`：原理（红黑树+就绪链表）、ET/LT模式
- 为什么epoll比select高效？（面试高频）

**多线程服务器**
- 每连接一个线程 vs 线程池
- 线程安全问题

### 推荐资源
- 书：《Linux高性能服务器编程》（游双）前8章 —— 这本书质量很高
- 视频：B站搜"Linux网络编程"

### 项目：多线程HTTP服务器 ⭐ 核心面试项目

```
阶段一（基础版）：
├── 监听端口，接受HTTP GET请求
├── 解析请求头，提取请求路径
├── 返回静态文件（HTML/图片）
├── 支持MIME类型判断
└── 每连接一个线程处理

阶段二（进阶版）：
├── 用epoll替代多线程（高并发）
├── 非阻塞IO + 事件驱动
├── 连接超时处理
└── 简单的HTTP响应码（200/404/500）
```

这个项目覆盖了：Socket编程、epoll、多线程、协议解析，是嵌入式物联网岗位的核心技能。

**字节面经相关考点**：
- TCP和UDP分别有什么特点？结合你的项目说一下
- epoll和select的区别？
- 如何实现高并发服务器？

## Task 2：单片机进阶（3月-6月）【必做】

### 学习内容

在寒假STM32基础上深入：

**ADC/DAC**
- ADC原理（逐次逼近型）
- 多通道ADC采集
- DAC输出模拟电压

**DMA（直接内存访问）**
- DMA原理（不经过CPU搬运数据）
- ADC + DMA：高速连续采样
- UART + DMA：高效数据发送

**PWM进阶**
- 电机调速（PWM占空比控制）
- 编码器读取（测速）

### 项目：多传感器数据采集系统
```
功能：
├── 多路ADC采集（光敏+热敏+电位器）
├── DMA高速传输ADC数据
├── 数据滤波（滑动平均）
├── 串口DMA上传到PC
└── OLED实时显示波形/数值
```
