第一阶段：C语言进阶 + Linux入门

2026年暑假（7-8月）· 每天4-6小时 · 共8周

📅 安排

周次内容重点
第1-4周C语言进阶指针、内存、链表、多文件编译
第5-8周Linux入门命令行、Makefile、Shell脚本

Task 1：C语言进阶 必做

核心突破点：

  • 指针：多级指针、函数指针、指针数组 vs 数组指针
  • 内存管理：malloc/free、内存泄漏、野指针、悬空指针
  • 结构体与链表：内存对齐、单/双向链表、循环链表
  • 位操作：设置/清除/翻转某一位（嵌入式高频考点）
  • 多文件编译：extern、static、编译链接过程

推荐：《C和指针》+ B站翁恺C语言进阶

🔨 项目：通讯录管理系统

功能要求：
├── 联系人增删改查（用链表实现）
├── 数据持久化（文件读写存储）
├── 多文件组织（main.c / contact.c / file.c）
└── 可扩展：排序、搜索、分类

Task 2：Linux环境搭建 必做

  • 安装 WSL2 + Ubuntu 22.04
  • 核心命令：ls cd grep find gcc make vim
  • Makefile：从单文件到多文件，变量、通配符、伪目标
  • Shell脚本：变量、条件、循环

🔨 项目：移植通讯录到Linux

任务：
├── 在Linux下编译运行通讯录管理系统
├── 编写完整Makefile
├── 写build.sh一键编译+运行+清理
└── 体验完整Linux开发流程

Task 3：GitHub习惯 有更好

注册GitHub，每天推代码。坚持两年，commit记录就是你的简历。

✅ 阶段检验

  • 手写双向链表增删改查（不看资料）
  • 解释 char *p vs char **p vs char (*p)[10]
  • Linux下用vim+gcc+Makefile完成项目
  • 写Shell脚本自动化构建流程
  • 解释内存泄漏原理

第二阶段：数据结构 + Linux系统编程

大二上学期（2026年9月-2027年1月）· 每天1-2小时

📅 安排

月份内容优先级
9月-11月数据结构必做
11月-1月Linux系统编程必做
贯穿全程LeetCode刷题尽量做
12月底提前网购STM32开发板（50-100元），寒假到货即用！

Task 1：数据结构 必做

必须手写实现：链表、栈、队列、哈希表、二叉树、堆

推荐：B站浙大数据结构（陈越）+ 《数据结构（C语言版）》严蔚敏

🔨 项目：手写哈希表 + 任务调度队列

Task 2：Linux系统编程 必做

  • 文件IO：open/read/write/lseek/close、文件描述符本质
  • 进程：fork（写时复制）、exec、wait、僵尸/孤儿进程
  • IPC：管道、共享内存、消息队列、信号
  • 线程：pthread、互斥锁、条件变量

🔨 项目：简易Shell解释器 ⭐

功能：
├── 内置命令：cd、exit、help
├── 外部命令执行（fork + exec）
├── 管道：ls | grep txt
├── 重定向：ls > output.txt
├── 后台执行：sleep 10 &
└── 信号处理：Ctrl+C不退出
字节面经考点：文件描述符是什么？进程间通信方法？fork原理？

✅ 阶段检验

  • 手写双向链表、栈、队列、哈希表
  • 解释fork执行过程，画父子进程图
  • Shell解释器能处理 ls -l | grep txt > out.txt
  • LeetCode累计50题

第三阶段：STM32 + RTOS入门

大二寒假（2027年1月-2月）· 每天4-6小时 · ⭐黄金期

🛒 购物清单（12月底买）

物品价格
STM32F103C8T6最小系统板~20元
ST-Link V2调试器~15元
杜邦线+面包板~10元
DHT11温湿度传感器~3元
0.96寸OLED屏（I2C）~8元
总计约60元

Task 1：STM32入门（前3周）必做

跟着正点原子/野火视频学：GPIO → UART串口 → 定时器 → 中断 → I2C → SPI

🔨 项目：串口调试助手 + 温湿度监测系统

温湿度系统：
├── DHT11定时采集
├── OLED实时显示
├── 串口上报到PC
└── 温度超阈值报警

Task 2：FreeRTOS入门（后2周）尽量做

核心：任务创建 → 信号量 → 队列 → 互斥锁 → 任务通知

🔨 项目：多任务温湿度系统

3个任务：
├── 采集任务：读DHT11，写入队列
├── 显示任务：从队列读数据，刷新OLED
└── 报警任务：监测温度，触发蜂鸣器
字节面试考了"FreeRTOS任务通知原理"

✅ 阶段检验

  • 独立接线烧录LED闪烁程序
  • 串口双向通信
  • 解释I2C协议（起始信号、地址帧、应答）
  • FreeRTOS 3个任务+队列通信
  • 温湿度系统完整运行

第四阶段：网络编程 + 单片机进阶

大二下学期（2027年2月-7月）· 每天1-2小时

📅 时间分配

网络编程（主线）70% ｜ 单片机进阶（副线）30%

Task 1：网络编程 必做

  • 基础：TCP/UDP区别、三次握手/四次挥手、字节序
  • Socket：TCP/UDP Socket编程、阻塞vs非阻塞IO
  • IO多路复用：select → poll → epoll（面试高频）
  • 多线程：每连接一线程 vs 线程池

🔨 项目：多线程HTTP服务器 ⭐

基础版：监听端口 → 解析HTTP → 返回静态文件
进阶版：epoll + 非阻塞IO + 高并发 + 超时处理
字节面经：TCP和UDP特点？epoll和select区别？

Task 2：单片机进阶 必做

ADC/DAC → DMA → PWM进阶（电机调速+编码器）

🔨 项目：多传感器数据采集系统

多路ADC采集 → DMA高速传输 → 数据滤波 → 串口DMA上传 → OLED显示

✅ 阶段检验

  • 手写TCP Server接受多客户端
  • 画TCP三次握手过程
  • HTTP服务器返回静态页面
  • epoll处理100+并发
  • DMA + ADC多通道采样

第五阶段：嵌入式Linux核心

大二暑假（2027年7月-8月）· 每天4-6小时

从零构建完整嵌入式Linux系统：上电 → U-Boot → 内核 → 文件系统 → 应用

🛒 硬件准备

正点原子IMX6ULL开发板（400-600元）—— 资料最全，社区活跃
预算有限可用QEMU虚拟机模拟（体验差一些）

8周安排 全必做

周次内容
1-2周交叉编译工具链 + U-Boot编译烧录
3-4周内核编译（make menuconfig）+ 设备树DTS
5-6周根文件系统构建（Buildroot + 手动构建）
7-8周应用层开发 + 完整系统调试

🔨 项目：开发板LED控制程序

设备树配置LED GPIO → 应用层控制 → 闪烁/呼吸灯 → 命令行交互

✅ 阶段检验

  • 解释交叉编译和本地编译区别
  • 修改U-Boot启动参数
  • make menuconfig裁剪内核
  • 修改设备树添加硬件描述
  • Buildroot构建最小文件系统
  • 完整走一遍启动流程并能讲清楚
