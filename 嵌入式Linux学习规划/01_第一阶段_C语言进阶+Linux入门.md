## Task 1：C语言进阶（第1-4周）【必做】

### 核心知识点

**指针（重中之重）**
- 指针变量本质、指针与数组的区别
- 多级指针（`char **argv` 的含义）
- 函数指针（回调函数的实现）
- 指针数组 vs 数组指针

**内存管理**
- 栈内存 vs 堆内存（malloc/free）
- 内存泄漏的危害与检测方法
- 野指针、悬空指针是什么、如何避免

**结构体与链表**
- 结构体内存对齐原理
- 单链表、双向链表、循环链表的实现
- 用链表做增删改查

**位操作**
- 按位与/或/异或/取反/移位
- 用位操作设置/清除/翻转某一位（嵌入式高频考点）

**多文件编译**
- 头文件与源文件的组织
- `extern`、`static`关键字
- 编译链接过程（预处理→编译→汇编→链接）

### 推荐资源
- 书：《C和指针》（Kenneth Reek）—— 重点看指针、数组、字符串章节
- 视频：B站翁恺C语言（进阶部分）

## Task 2：Linux环境搭建（第5-8周）【必做】

### 环境搭建
- 安装 **WSL2 + Ubuntu 22.04**（不用双系统，Windows下最省心的方案）
- 安装工具：`gcc`、`make`、`vim`/`neovim`、`git`

### 核心命令（必须熟练）

**文件操作**：`ls`、`cd`、`cp`、`mv`、`rm`、`cat`、`head`、`tail`
**搜索**：`find`、`grep`
**权限**：`chmod`、`chown`
**进程**：`ps`、`top`、`kill`
**编译**：`gcc`、`make`

### Makefile（重点）

从最简单的开始：
```makefile
# 单文件
main: main.c
    gcc -o main main.c

# 多文件
main: main.o contact.o file.o
    gcc -o main main.o contact.o file.o

%.o: %.c
    gcc -c $< -o $@
```

逐步加入：变量、通配符、伪目标（.PHONY）、依赖规则。

### Shell脚本基础

```bash
#!/bin/bash
# 变量、条件、循环
for file in *.c; do
    echo "编译: $file"
    gcc -c "$file"
done
```
