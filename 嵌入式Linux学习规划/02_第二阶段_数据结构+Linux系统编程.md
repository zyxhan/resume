## Task 1：数据结构（9月-11月）【必做】

### 必须掌握的数据结构

**线性结构**
- 单向链表、双向链表、循环链表（手写实现）
- 栈（数组实现 + 链表实现）
- 队列（循环队列、优先队列）

**树形结构**
- 二叉树（前/中/后序遍历）
- 二叉搜索树（插入、查找、删除）
- 堆（最小堆/最大堆）

**哈希表**
- 哈希函数设计
- 冲突解决：链地址法、开放寻址法
- 负载因子与扩容

### 学习方法
- 每种数据结构先理解原理，再手写实现
- 不需要像ACM那么深，但要能**不看资料手撕代码**
- 这是嵌入式面试手撕题的基础

### 推荐资源
- 书：《数据结构（C语言版）》严蔚敏
- 视频：B站浙大数据结构（陈越、何钦铭）

### 项目1：手写哈希表
```
功能：
├── 插入、查找、删除键值对
├── 链地址法解决冲突
├── 自动扩容（负载因子 > 0.75 时翻倍）
└── 用字符串作为key
```

### 项目2：任务调度队列
```
功能：
├── 优先级队列（堆实现）
├── 支持插入任务、取出最高优先级任务
├── 支持任务取消
└── 打印当前所有任务及优先级
```

## Task 2：Linux系统编程（11月-1月）【必做】

### 核心知识点

**文件IO**
- 系统调用：`open`/`read`/`write`/`lseek`/`close`
- 文件描述符的本质（内核文件表项的索引）
- 标准IO vs 系统IO的区别
- 文件锁（`fcntl`）

**进程**
- `fork()` 的本质（写时复制）
- `exec` 族函数（替换进程映像）
- `wait`/`waitpid`（回收子进程）
- 僵尸进程和孤儿进程

**进程间通信（IPC）**
- 管道（匿名管道 + 命名管道）
- 共享内存（最快的IPC方式）
- 消息队列
- 信号（`signal`/`sigaction`）

**线程基础**
- `pthread_create`/`pthread_join`
- 互斥锁（`pthread_mutex`）
- 条件变量（`pthread_cond`）

### 推荐资源
- B站：搜"Linux系统编程"（黑马程序员/正点原子）
- 书：《Linux程序设计》（Beginning Linux Programming）

### 项目：简易Shell解释器 ⭐ 面试高频项目

```
功能要求：
├── 读取用户输入的命令
├── 支持内置命令：cd、exit、help
├── 支持外部命令执行（fork + exec）
├── 支持管道：ls | grep txt
├── 支持重定向：ls > output.txt、cat < input.txt
├── 支持后台执行：sleep 10 &
└── 支持信号处理：Ctrl+C 不退出Shell
```

这个项目综合了：进程创建、管道通信、信号处理、文件IO，是面试中最常问的项目之一。

**字节面经相关考点**：
- Linux中文件描述符是什么？
- Linux中有哪些进程间通信方法？
- fork的原理是什么？

## Task 3：LeetCode刷题（贯穿全学期）【尽量做】

### 刷题策略
- 每天1-2题，从**简单难度**开始
- 只用C语言写（巩固C功底）
- 按专题刷：数组 → 链表 → 栈/队列 → 字符串 → 树

### 推荐顺序
1. 两数之和、反转字符串、合并有序数组（数组基础）
2. 反转链表、合并链表、删除节点（链表操作）
3. 有效括号、最小栈（栈应用）
4. 二叉树遍历、最大深度（树基础）

### 学期目标
- 累计完成 50 题（简单为主）
- 重点是理解思路，不是追求数量
