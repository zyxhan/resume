## Task 1：简历准备（2月）

### GitHub整理
- 每个项目仓库写好README：
  - 项目介绍（做什么的）
  - 架构图/流程图
  - 技术栈
  - 如何运行
- 精选2-3个项目置顶展示

### 简历要点
```
技术栈：
├── C/C++（熟练），Python/Shell（了解）
├── Linux系统编程（进程/线程/IPC/网络）
├── 嵌入式Linux（U-Boot/内核/设备树/驱动）
├── STM32 + FreeRTOS
├── 网络编程（TCP/UDP/Socket/epoll）
└── 通信协议（UART/I2C/SPI）

项目经历（写2-3个）：
├── 智能家居网关（综合项目，重点讲）
├── 多线程HTTP服务器（网络编程，体现系统能力）
└── 简易Shell解释器（系统编程，体现底层理解）

其他：
├── GitHub地址（有持续提交记录）
├── 竞赛奖项（如有）
└── GPA（前30%就写）
```

### 简历避坑
- 不要写"精通"，写"熟悉"/"掌握"
- 不要堆砌技术，写真正做过的
- 项目描述要有数据：支持XX并发、延迟XXms

## Task 2：面试八股文专项（2月-3月）

### 字节嵌入式面试高频考点

**C语言（P0 - 必须精通）**
- 指针与数组的区别
- `volatile`关键字（嵌入式必问）
- `static`修饰局部/全局/函数的效果
- 结构体内存对齐（为什么、怎么算）
- 宏的副作用（`#define MAX(a,b)` 的坑）
- 函数指针 vs 指针函数
- `malloc`分配失败的处理（外部碎片问题）

**操作系统（P0）**
- 进程 vs 线程
- 进程调度算法
- 内存管理（分页/分段/虚拟内存）
- 文件描述符是什么
- 死锁的条件和预防

**Linux内核（P1）**
- 进程间通信方式
- DMA内存分配原理
- Cache一致性问题
- 中断处理机制（顶半部/底半部）

**RTOS（P1）**
- FreeRTOS任务通知原理
- 中断分段的原理和原因
- RTOS vs Linux的区别

**网络（P1）**
- TCP vs UDP（结合项目说）
- TCP三次握手/四次挥手
- epoll vs select

**数据结构（P1）**
- 堆 vs 栈（作为数据结构）
- 数组 vs 链表
- 环形缓冲区原理

### 手撕代码（面试常考）
- 反转链表（字节考过）
- 给指针分配内存函数
- 字符串反转/拷贝
- 实现一个简单的内存分配器

### 推荐资源
- 牛客网嵌入式面经（必看）
- 小林coding（图解网络/操作系统，免费）
- CSDN嵌入式面试总结帖

## Task 3：LeetCode冲刺（2月-面试前）【尽量做】

### 目标：累计80-100题

### 重点题型
- 数组/字符串操作
- 链表操作（反转、合并、找环）
- 栈/队列应用
- 简单动态规划（爬楼梯、最长子序列）

- 写完后自己测一遍

## Task 5：持续优化

### 如果面试不顺怎么办？
- 分析是简历问题还是面试问题
  - 简历投了没回音 → 优化简历/项目
  - 一面挂了 → 八股文/手撕代码不够
  - 二面挂了 → 项目讲不清楚
- 针对性补强，不要盲目海投

### 面试心态
- 前几次面试紧张是正常的
- 面试是双向选择，不是你求着公司
- 每一次面试都是学习机会
- 被拒不代表你不行，只是方向不匹配
